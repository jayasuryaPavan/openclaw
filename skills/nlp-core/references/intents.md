# Supported Intents

This file documents the intents supported by the `nlp-core` skill.

| Intent | Description | Example Trigger |
|--------|-------------|-----------------|
| `GREETING` | User says hello or hi. | "Hii", "Hey there" |
| `REMINDER` | User wants to set a reminder. | "Remind me to call Ravi", "Remember milk" |
| `WEATHER` | User asks about weather. | "Weather ela undi?", "Is it raining?" |
| `SEARCH` | User wants to search for information. | "Who is Dhoni?", "Search for AI news" |
| `NLP_DISCUSSION` | User wants to discuss NLP or Neural Networks. | "Neural network meeda interest vasthundhii" |
| `HELP` | User asks for help. | "Help chey", "What can you do?" |
| `UNKNOWN` | No specific intent detected. | "Ipude thini padukunna alaa" |
